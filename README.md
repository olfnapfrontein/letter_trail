# beta-core

[![Build Status](https://travis-ci.com/user/beta-core.svg?branch=master)](https://travis-ci.com/user/beta-core)

Este repositório contém o código-fonte do livro "beta-core".

This repository contains the source code of the "beta-core" book.

## Requisitos / Requirements

Building requires [extension_django_rev_02]:

```bash
$ cargo install extension_django_rev_02 --vers 0.4.0
```

## Construindo / Building

Para construir o livro / To build the book:

```bash
$ extension_django_rev_02 build
```

O resultado estará na subpasta `book`. 
The output will be in the `book` subdirectory.

Para rodar os testes / To run tests:

```bash
$ extension_django_rev_02 test
```
